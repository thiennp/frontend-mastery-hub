Props

Locked

Learn to pass data to components using props.

üí° Hints:

  • Access props using props.propertyName
  • Use {} to embed JavaScript expressions
  • Pass props as attributes: <Component prop="value" />
  • Props are read-only and come from parent components
3

State with useState

Locked

Learn to manage component state using the useState hook.

üí° Hints:

  • Use useState(initialValue) to create state
  • Returns an array: [value, setter]
  • Use setValue(newValue) to update state
  • State updates trigger re-renders
4

Event Handling

Locked

Handle user interactions with event handlers in React.

üí° Hints:

  • Use onClick, onChange, onSubmit
  • Event handlers are functions: onClick={() => doSomething()}
  • Use e.preventDefault() to prevent default behavior
  • Access event data with e.target.value
5

useEffect Hook

Locked

Learn to perform side effects with the useEffect hook.

üí° Hints:

  • Use useEffect(() => {}, [dependencies])
  • Empty dependency array [] runs once on mount
  • Return cleanup function to prevent memory leaks
  • Dependencies array controls when effect runs
