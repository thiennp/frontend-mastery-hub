## üìñ Modules

### 1. **Angular Introduction** (3-4 hours)
- **Angular Overview**: What is Angular and why use it
- **Angular CLI**: Setting up and using Angular CLI
- **Project Structure**: Understanding Angular project organization
- **TypeScript Integration**: TypeScript in Angular applications
- **Development Environment**: IDE setup and development tools

### 2. **TypeScript Fundamentals** (2-3 hours)
- **TypeScript Basics**: Types, interfaces, and classes
- **Advanced Types**: Generics, unions, and intersections
- **Decorators**: Understanding Angular decorators
- **Modules and Imports**: ES6 modules and TypeScript
- **Type Safety**: Compile-time type checking

### 3. **Components and Templates** (3-4 hours)
- **Component Architecture**: Understanding Angular components
- **Template Syntax**: Angular template syntax and expressions
- **Data Binding**: Property, event, and two-way binding
- **Component Lifecycle**: Lifecycle hooks and component phases
- **Component Communication**: Parent-child component communication

### 4. **Services and Dependency Injection** (2-3 hours)
- **Service Architecture**: Creating and using services
- **Dependency Injection**: Angular's DI system
- **Injectable Services**: Making services injectable
- **Service Providers**: Providing services in modules
- **Singleton Services**: Understanding service scope

### 5. **Directives and Pipes** (2-3 hours)
- **Built-in Directives**: *ngIf, *ngFor, *ngSwitch, and more
- **Custom Directives**: Creating custom structural and attribute directives
- **Built-in Pipes**: Date, currency, number, and other pipes
- **Custom Pipes**: Creating custom pipes for data transformation
- **Pipe Purity**: Understanding pure and impure pipes
