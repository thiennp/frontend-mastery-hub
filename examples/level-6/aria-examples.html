ARIA Examples - Level 6

These examples demonstrate ARIA attributes and their proper usage for accessibility.

1. Basic ARIA Roles

This example shows common ARIA roles and their usage.

<!-- Button role -->
<div role="button" tabindex="0" onclick="handleClick()">
    Custom Button
</div>

<!-- Alert role -->
<div role="alert" aria-live="assertive">
    This is an important alert message
</div>

<!-- Status role -->
<div role="status" aria-live="polite">
    Status update: Operation completed
</div>

<!-- Progress bar -->
<div role="progressbar" 
     aria-valuemin="0" 
     aria-valuemax="100" 
     aria-valuenow="75"
     aria-label="Loading progress">
    75%
</div>

2. Dropdown Menu with ARIA

This example demonstrates a dropdown menu with proper ARIA attributes.

<div class="dropdown">
    <button class="dropdown-trigger" 
            aria-expanded="false" 
            aria-haspopup="menu" 
            aria-controls="menu1">
        Actions
    </button>
    <ul class="dropdown-menu" 
        id="menu1" 
        role="menu" 
        aria-labelledby="trigger1"
        aria-hidden="true">
        <li role="none">
            <a href="#" role="menuitem" tabindex="-1">Edit</a>
        </li>
        <li role="none">
            <a href="#" role="menuitem" tabindex="-1">Delete</a>
        </li>
    </ul>
</div>

  • Edit
  • Delete

3. Modal Dialog with ARIA

This example shows a modal dialog with proper ARIA attributes for accessibility.

<div class="modal" 
     role="dialog" 
     aria-modal="true" 
     aria-labelledby="modal-title"
     aria-describedby="modal-description"
     tabindex="-1"
     aria-hidden="true">
    
    <div class="modal-content">
        <h2 id="modal-title">Confirm Action</h2>
        <p id="modal-description">
            Are you sure you want to perform this action?
        </p>
        <button onclick="closeModal()">Cancel</button>
        <button onclick="confirmAction()">Confirm</button>
    </div>
</div>

4. Form with ARIA Validation

This example demonstrates form validation with ARIA attributes.

<form>
    <div>
        <label for="email">Email *</label>
        <input type="email" 
               id="email" 
               name="email" 
               required
               aria-invalid="false"
               aria-describedby="email-error email-help">
        <div id="email-help">We'll never share your email</div>
        <div id="email-error" 
             role="alert" 
             aria-live="polite"></div>
    </div>
    
    <button type="submit">Submit</button>
</form>

Live Demo:

5. Live Regions for Dynamic Content

This example shows how to use live regions to announce dynamic content changes.

<!-- Polite live region -->
<div aria-live="polite" id="status-updates">
    Status updates will appear here
</div>

<!-- Assertive live region -->
<div aria-live="assertive" id="error-messages">
    Error messages will appear here
</div>

<!-- Status role -->
<div role="status" aria-live="polite">
    Current status: Ready
</div>

<!-- Alert role -->
<div role="alert" aria-live="assertive">
    No alerts at this time
</div>

6. Tab Interface with ARIA

This example demonstrates a tab interface with proper ARIA attributes.

<div role="tablist" aria-label="Settings">
    <button role="tab" 
            aria-selected="true" 
            aria-controls="panel1"
            id="tab1">
        General
    </button>
    <button role="tab" 
            aria-selected="false" 
            aria-controls="panel2"
            id="tab2">
        Advanced
    </button>
</div>

<div role="tabpanel" 
     id="panel1" 
     aria-labelledby="tab1">
    General settings content...
</div>

<div role="tabpanel" 
     id="panel2" 
     aria-labelledby="tab2"
     aria-hidden="true">
    Advanced settings content...
</div>

7. Progress Indicators with ARIA

This example shows different types of progress indicators with ARIA attributes.

<!-- Progress bar -->
<div role="progressbar" 
     aria-valuemin="0" 
     aria-valuemax="100" 
     aria-valuenow="75"
     aria-labelledby="progress-label">
    <div class="progress-fill" style="width: 75%"></div>
</div>
<div id="progress-label">75% complete</div>

<!-- Step progress -->
<nav aria-label="Progress">
    <ol class="step-progress">
        <li class="step completed" aria-current="false">
            <span class="step-number">1</span>
            <span class="step-label">Account</span>
        </li>
        <li class="step current" aria-current="true">
            <span class="step-number">2</span>
            <span class="step-label">Profile</span>
        </li>
        <li class="step" aria-current="false">
            <span class="step-number">3</span>
            <span class="step-label">Review</span>
        </li>
    </ol>
</nav>

8. Tooltips with ARIA

This example demonstrates accessible tooltips using ARIA attributes.

<div class="tooltip-container">
    <button aria-describedby="tooltip1" 
            aria-expanded="false"
            onclick="toggleTooltip('tooltip1')">
        Hover for more info
    </button>
    <div class="tooltip" 
         id="tooltip1" 
         role="tooltip" 
         aria-hidden="true">
        This is additional information about the button.
    </div>
</div>
