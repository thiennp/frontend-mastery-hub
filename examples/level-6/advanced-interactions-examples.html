Advanced Interactions Examples - Level 6

These examples demonstrate advanced accessible interactions and complex widgets.

1. Accessible Modal Dialog

This example shows a fully accessible modal dialog with focus trapping and keyboard navigation.

<div class="modal" 
     role="dialog" 
     aria-modal="true" 
     aria-labelledby="modal-title"
     aria-describedby="modal-description"
     tabindex="-1"
     aria-hidden="true">
    
    <div class="modal-content">
        <h2 id="modal-title">Confirm Action</h2>
        <p id="modal-description">
            Are you sure you want to perform this action?
        </p>
        <button onclick="closeModal()">Cancel</button>
        <button onclick="confirmAction()">Confirm</button>
    </div>
</div>

2. Accessible Dropdown Menu

This example demonstrates a dropdown menu with full keyboard support and ARIA attributes.

<div class="dropdown">
    <button class="dropdown-trigger" 
            aria-expanded="false" 
            aria-haspopup="menu" 
            aria-controls="menu1"
            onclick="toggleDropdown('menu1')">
        Actions
    </button>
    <ul class="dropdown-menu" 
        id="menu1" 
        role="menu" 
        aria-labelledby="trigger1"
        aria-hidden="true">
        <li role="none">
            <a href="#" role="menuitem" tabindex="-1">Edit</a>
        </li>
        <li role="none">
            <a href="#" role="menuitem" tabindex="-1">Delete</a>
        </li>
    </ul>
</div>
  • Edit
  • Delete

4. Accessible Tab Interface

This example demonstrates a tab interface with proper ARIA attributes and keyboard navigation.

<div class="tabs">
    <ul class="tab-list" role="tablist">
        <li role="presentation">
            <button role="tab" 
                    aria-selected="true" 
                    aria-controls="tabpanel1"
                    id="tab1"
                    onclick="switchTab('tab1', 'tabpanel1')">
                Tab 1
            </button>
        </li>
        <li role="presentation">
            <button role="tab" 
                    aria-selected="false" 
                    aria-controls="tabpanel2"
                    id="tab2"
                    onclick="switchTab('tab2', 'tabpanel2')">
                Tab 2
            </button>
        </li>
    </ul>
    
    <div role="tabpanel" 
         id="tabpanel1" 
         aria-labelledby="tab1">
        Content for tab 1...
    </div>
    
    <div role="tabpanel" 
         id="tabpanel2" 
         aria-labelledby="tab2"
         aria-hidden="true">
        Content for tab 2...
    </div>
</div>
Content for tab 1...

5. Accessible Carousel

This example shows a carousel with proper ARIA attributes and keyboard navigation.

<div class="carousel" role="region" aria-label="Image carousel">
    <div class="carousel-container">
        <div class="carousel-track">
            <div class="carousel-slide">
                <img src="image1.jpg" alt="Image 1">
            </div>
            <div class="carousel-slide">
                <img src="image2.jpg" alt="Image 2">
            </div>
        </div>
    </div>
    
    <div class="carousel-controls">
        <button onclick="previousSlide()" aria-label="Previous slide">❮</button>
        <button onclick="nextSlide()" aria-label="Next slide">❯</button>
    </div>
    
    <div class="carousel-indicators" role="tablist">
        <button class="carousel-indicator active" 
                role="tab" 
                aria-selected="true"
                onclick="goToSlide(0)"></button>
        <button class="carousel-indicator" 
                role="tab" 
                aria-selected="false"
                onclick="goToSlide(1)"></button>
    </div>
</div>

6. Drag and Drop Interface

This example demonstrates a drag and drop interface with proper ARIA attributes.

<div class="drag-drop" 
     role="region" 
     aria-label="Drop zone"
     ondrop="handleDrop(event)" 
     ondragover="handleDragOver(event)" 
     ondragenter="handleDragEnter(event)" 
     ondragleave="handleDragLeave(event)">
    Drop items here
</div>

<div class="draggable" 
     draggable="true" 
     role="button" 
     tabindex="0"
     ondragstart="handleDragStart(event)" 
     onkeydown="handleDragKeydown(event)">
    Draggable Item 1
</div>

8. Accessible Notification System

This example demonstrates a notification system with proper ARIA attributes and announcements.

     role="alert"

    Notification message
</div>

<button onclick="showNotification('Success!', 'success')">

</button>
