CSS Specificity Examples

These elements demonstrate how CSS specificity works. The most specific rule wins!

/* Specificity: 0,0,0,1 */
div { color: black; }

/* Specificity: 0,0,1,0 */
.container { color: blue; }

/* Specificity: 0,0,1,1 */
div.container { color: green; }

/* Specificity: 0,1,0,0 */
#main { color: red; }

This div has class "container" (specificity: 0,0,1,1) - should be green

This div has id "main" (specificity: 0,1,0,1) - should be purple

This div has class "container item" (specificity: 0,0,2,0) - should be orange

This div has class "container item active" (specificity: 0,0,3,0) - should be yellow

CSS Custom Properties Examples

This page uses CSS custom properties for consistent theming:
