Asynchronous JavaScript Examples

Level 4 - Frontend Mastery Hub

1. Callbacks

Callbacks are functions passed as arguments to other functions and executed at a later time.

// Basic callback example function fetchData(callback) { setTimeout(() => { const data = { message: "Data fetched successfully!" }; callback(null, data); }, 1000); } // Using the callback fetchData((error, data) => { if (error) { console.error("Error:", error); } else { console.log("Success:", data); } });

2. Promises

Promises provide a cleaner way to handle asynchronous operations and avoid callback hell.

// Promise example function fetchDataPromise() { return new Promise((resolve, reject) => { setTimeout(() => { const success = Math.random() > 0.3; // 70% success rate if (success) { resolve({ message: "Promise resolved successfully!" }); } else { reject(new Error("Promise rejected!")); } }, 1500); }); } // Using the promise fetchDataPromise() .then(data => console.log("Success:", data)) .catch(error => console.error("Error:", error));

3. Async/Await

Async/await provides a more readable way to work with promises, making asynchronous code look synchronous.

// Async/await example async function fetchDataAsync() { try { const data = await fetchDataPromise(); console.log("Success:", data); return data; } catch (error) { console.error("Error:", error); throw error; } } // Using async/await fetchDataAsync();

4. Promise Chaining

Chain multiple asynchronous operations together using promises.

// Promise chaining example function step1() { return new Promise(resolve => { setTimeout(() => resolve("Step 1 completed"), 500); }); } function step2(data) { return new Promise(resolve => { setTimeout(() => resolve(data + " → Step 2 completed"), 500); }); } function step3(data) { return new Promise(resolve => { setTimeout(() => resolve(data + " → Step 3 completed"), 500); }); } // Chaining promises step1() .then(step2) .then(step3) .then(result => console.log("Final result:", result)) .catch(error => console.error("Error in chain:", error));

5. Parallel Execution

Execute multiple asynchronous operations in parallel using Promise.all() and Promise.allSettled().

// Parallel execution example async function fetchMultipleData() { const promises = [ fetchDataPromise(), fetchDataPromise(), fetchDataPromise() ]; try { // Wait for all promises to resolve const results = await Promise.all(promises); console.log("All promises resolved:", results); } catch (error) { console.error("One or more promises failed:", error); } // Or use Promise.allSettled to get all results const settledResults = await Promise.allSettled(promises); console.log("All promises settled:", settledResults); }

6. Error Handling

Proper error handling in asynchronous JavaScript using try/catch and promise error handling.

// Error handling example async function robustAsyncFunction() { try { const data = await fetchDataPromise(); console.log("Success:", data); return data; } catch (error) { console.error("Caught error:", error.message); // Handle error gracefully return { error: true, message: "Failed to fetch data" }; } } // Promise error handling fetchDataPromise() .then(data => { console.log("Success:", data); return data; }) .catch(error => { console.error("Promise error:", error.message); return { error: true, message: "Promise failed" }; }) .finally(() => { console.log("Operation completed (success or failure)"); });

7. Real-world Example: Weather API

A practical example of asynchronous JavaScript using a weather API.

// Weather API example async function getWeatherData(city) { try { const response = await fetch(`https://api.openweathermap.org/data/2.5/weather?q=${city}&appid=YOUR_API_KEY`); if (!response.ok) { throw new Error(`HTTP error! status: ${response.status}`); } const data = await response.json(); return data; } catch (error) { console.error("Error fetching weather data:", error); throw error; } } // Using the weather function getWeatherData("London") .then(weather => console.log("Weather data:", weather)) .catch(error => console.error("Failed to get weather:", error));

8. Performance Monitoring

Monitor the performance of asynchronous operations.

// Performance monitoring example async function measureAsyncPerformance() { const startTime = performance.now(); try { const result = await fetchDataPromise(); const endTime = performance.now(); const duration = endTime - startTime; console.log(`Operation completed in ${duration.toFixed(2)}ms`); return { result, duration }; } catch (error) { const endTime = performance.now(); const duration = endTime - startTime; console.log(`Operation failed after ${duration.toFixed(2)}ms`); throw error; } }
